# 要件定義書

## 1. 背景

- Turn Bomber をスマホ/PCで手軽に遊べるようにし、ネットワーク不安定時でも継続プレイできる状態にする。

## 2. 目的

- アプリを PWA 化し、インストール可能にする。
- 一度読み込んだ後は、オフラインでもゲーム起動・プレイできるようにする。

## 3. 機能要件

1. Web App Manifest を提供すること。
2. Service Worker を導入し、アプリの必要静的アセットをキャッシュすること。
3. 盤面描画に必要な外部依存（Pixi CDN モジュール）をオフライン再利用できること。
4. ブラウザで「ホーム画面に追加/インストール」が可能であること。
5. 既存ゲームロジック（同時入力・同時解決）を壊さないこと。

## 4. 非機能要件

- 初回オンライン訪問後、次回以降の再訪問時にオフライン起動可能であること。
- 既存テスト (`npm test`) が通過すること。
- README / docs を更新し、運用手順を明記すること。

## 5. 対象外

- ネイティブアプリ化
- サーバーサイド API 追加
